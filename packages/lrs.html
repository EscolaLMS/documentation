# Install

  1. get package from composer composer require escolalms/lrs
  2. run the seeder php artisan db:seed --class="EscolaLms\Lrs\Database\Seeders\LrsSeeder"
  3. make sure that Response Headers are not overwritten by any layer, the /api/cmi5/** should response with
x-experience-api-version: 1.0.3

# Testing

  1. Download cmi5-demo (opens new window) and run it with static file server - npm run serve (opens new window) or php -S localhost:8000 (opens new window) is good enough
  2. Generate fetch params (opens new window) for a course id calling /api/cmi5/courses/{id} endpoint
