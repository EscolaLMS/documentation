# Installation from source

There are 3 main repositories to install from source, each of them have a readme.md file on github main page with instruction how to install from source

  1. API (opens new window) - the collection of backend components attached to database and other services. Laravel Application (opens new window).
  2. Admin Panel (opens new window) - Content and Learning Management System, requires API component. Ant Design Pro Application (opens new window)
  3. Demo (opens new window) - implementation of user Frontend, requires API component. TypeScript Create React Application (opens new window)

One install API as standalone Laravel Application, and Admin Panel and Demo as standard React Application.

This article will not cover how to perform above.
