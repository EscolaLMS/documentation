ALL endpoints are describes in open api (previously know as swagger) format.

  • The raw json that combines all packages latest version will be here https://github.com/EscolaLMS/API/blob/gh-pages/api-docs.json (opens new window). This file can be consumed like a standard openapi/swagger file.
  • Development version of the api that Swagger UI is attached to https://api-stage.escolalms.com/api/documentation#/ (opens new window) NOTE this might not be stable version
  • Main documentation is stored at api-docs.wellms.io (opens new window)

Also each package has it's own API Description in their github pages like,

  • eg Courses documentation would be here https://escolalms.github.io/Courses/ (opens new window)
